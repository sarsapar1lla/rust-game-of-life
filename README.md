# Game of Life

Rust implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

## Test

Run tests using cargo:

```bash
$ cargo test
```

## Build

Build a web assembly distributable:

```bash
$ cargo build
```

## Run locally

The app can be run locally using trunk:

```bash
$ trunk serve --open
```
